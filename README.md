# PhoneWheel — PC App

Принимает данные от Android-приложения PhoneWheel по WebSocket (Wi-Fi или USB)
и выводит их в виртуальный джойстик через **vJoy**, чтобы Assetto Corsa, F1 и
iRacing видели телефон как обычный контроллер.

## Что нужно установить один раз

1. **vJoy** — драйвер виртуального джойстика.
   Скачать: https://sourceforge.net/projects/vjoy/ (бесплатный, открытый проект).
   После установки открой **Configure vJoy** и убедись, что включено
   устройство **#1** хотя бы с осями X/Y/Z и 5+ кнопками.
2. Для **USB**-режима — Android **platform-tools** (`adb.exe`) и путь к нему
   должен быть в `PATH`. Также включи в телефоне Настройки → Для
   разработчиков → **USB-отладка**.

## Запуск

Запусти `PhoneWheelPC.exe`. Готовые сборки появляются автоматически в вкладке
**Actions** репозитория на GitHub после каждого изменения кода (артефакт
`PhoneWheelPC-win64`).

- Если в строке "Сервер" написано про отсутствие доступа — нажми
  **Настроить доступ** один раз (запросит права администратора через UAC,
  дальше так делать не нужно).
- Для USB — нажми **Настроить USB (adb reverse)** после подключения телефона
  кабелем.
- Для Wi-Fi — просто убедись, что телефон и PC в одной сети, и введи IP PC в
  приложении на телефоне.

## Как работает привязка кнопок

Выбери шаблон игры (Assetto Corsa / F1 / iRacing) в выпадающем списке — это
не меняет ничего на телефоне, только задаёт, на какую кнопку vJoy будет
приходить нажатие "Передача вверх", "Ручник" и т.д. Цифры можно менять и
сохранять отдельно для каждого шаблона.

Дальше — **один раз** в самой игре: Controls/Управление → найди нужное
действие (например "Gear Up") → нажми назначенную кнопку vJoy на телефоне →
игра запомнит привязку. Руль и педали (газ/тормоз) обычно определяются играми
автоматически как оси X/Y/Z виртуального джойстика.

## Известные ограничения

- vJoy — это кернел-драйвер, его ставит сам пользователь, приложение его не
  устанавливает автоматически.
- Если `vJoyInterface.dll` не находится автоматически, скопируй её из папки
  установки vJoy (`C:\Program Files\vJoy\x64\`) рядом с `PhoneWheelPC.exe`.
- USB-режим требует ручной установки platform-tools — это ограничение adb, не
  приложения.
